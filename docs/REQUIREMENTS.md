# 产品需求

- 纯一战题材，单局 50v50，单场最多 100 名参战玩家。
- 官方固定开放两个战场，并允许玩家租赁受控社区战场。
- 四兵种：突击兵、医疗兵、支援兵、侦察兵。
- 首发模式为大战役推进模式。
- 首发载具内容为 2 辆坦克、1 辆装甲车、固定重机枪和野战炮。
- 载具模型、骨骼、动画、座位和模型挂载仅由 ModelEngine 负责，基础实体和技能仅由 MythicMobs 负责。
- 首发不制作飞机、骑兵、海战、巨兽或完全动态倒塌。
- 不出售战斗优势；外观不得提供战斗属性。
- 租赁者不得上传插件。
- 商业服经济、等级、战绩和资产与 Euro WarSim 完全隔离。
